# Specification: Tic-Tac-Toe Game

## 目標
開發一個可以在瀏覽器上玩的井字遊戲（Tic-Tac-Toe）。

## 功能需求
1. 玩家可以點擊 3x3 的格子來下 X 或 O。
2. 玩家輪流下棋，X 先手。
3. 每下一步自動判斷是否有玩家獲勝，或是否平手。
4. 顯示「X 贏了」、「O 贏了」或「平手」。
5. 提供一個「重新開始」按鈕。

## 技術需求
- 使用 HTML、CSS、JavaScript（純前端，不用框架）。
- 介面要簡潔，中央顯示 3x3 棋盤。
- 每次推送（push）到 main 分支後，GitHub Actions 自動：
  - 執行程式碼格式檢查（例如 Prettier）
  - 自動部署到 GitHub Pages

## 測試需求
- 每當玩家按下格子後，格子不能被重複點擊。
- 勝利條件（橫列、直列、對角線）正確判斷。
- 平手情況正確顯示。
